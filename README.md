# Zomato Restaurant Expansion Analysis

This project analyzes 9,551 restaurants across multiple countries to identify potential expansion opportunities for Zomato.

## Objective

Identify countries and cities where new restaurants could be opened based on:

• Competition levels  
• Customer ratings  
• Pricing patterns  
• Service availability (online delivery, table booking)

## Tools Used

• Excel / Spreadsheets  
• Pivot Tables  
• Lookup Functions  
• Data Cleaning  
• Dashboard Design  

## Key Insights

• Australia, Canada, and Qatar emerged as promising expansion markets  
• Cities like Balingup, Montville, Consort, Yorkton, and Doha show opportunity  
• Online delivery and table booking correlate with higher ratings  
• Price and ratings show weak correlation

## Dashboard

An interactive Excel dashboard built to visualize:

• Restaurant distribution  
• Ratings  
• Pricing trends  
• Market opportunities
